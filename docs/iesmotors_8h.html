◆ setDutyCycle()

void setDutyCycle ( uint8_t  pin,
uint8_t  value 
)

Set the Duty Cycle object.

Sets duty-cycle at pin PD5 or PD6 which defined in the with 'LEFT_SIDE' and 'RIGHT_SIDE' to a value (0 - 255 = 0% - 100%). Good Defined values are 'SPEED_HALF' and 'SPEED_MAX'.

Parameters
pinSet to 'LEFT_SIDE' or 'RIGHT_SIDE'
value0-255
Attention
Timer0 needs to be setup before usage.
The required pins also need to be set as output first.
